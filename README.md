# Train-Reservation-Sysetm
A train reservation system as a Client-Server application using basic linux system calls.
It has three actors : Admin, User(individual) and Agent.

Admin can:
  - Add a train.
  - Delete a train.
  - Change no of seats.

Agent & user can:
  - Book tickets.
  - View bookings.
  - Cancel bookings.
  - Update bookings.
